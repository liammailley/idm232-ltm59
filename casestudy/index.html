OVERVIEW

This class focused around database connections, PHP, and MySql. The purpose of this final project was to demonstrate understanding of PHP through the construction of a recipe site.
Users can view a collection of recipes, and can search for specific ones by using a search bar. An Admin section is also included, where one can edit individual recipes, create new recipes, and delete recipes. Finally, a section that allows for sorting by category is included.

CONTEXT AND CHALLENGE

BACKGROUND

Before I could begin using this new language to build this site, I needed to figure out how to display these recipes.
I eventually settled on a soft color palatte and a choice to brand the site with my own name.

You can check out my beta here.

By my beta, however, I had decided that I wanted to focus on something I could rework into my portfolio site, as I hadn't updated it in a while.

THE PROBLEM

My portfolio's way of displaying my work was clunky. It consisted of 4 different buttons for 4 different categories, each of which opened into a seperate page that displayed the projects for that category. It was functional, but not intuitive. Forcing the user to jump from page to page gets overwhelming. Additionally, I wanted my portfolio to reflect my level of comfortability with HTML, CSS, and Javascript.

GOALS AND OBJECTIVES

My goal was simple: find a way to condense all of the page-jumping content of my portfolio into a single group of microinteractions on a single page.

PROCESS AND INSIGHT

When I brought this idea up to my teacher, he suggested creating a variation on an image gallery. Looking through CodePen, I was able to find a very smooth example to base my code off of.

From there, I began creating special hover states that allowed for text to appear overtop of the image when it was hovered. I wanted this bit to feel seamless, so I paid special care to the animation times in order for it to all feel like one smooth animation.

Next, I created 5 modals. Each would only be visible when its category had been selected from the gallery. I transferred the content from my portfolio, and was finally able to access all of my work from one microinteraction group.

THE RESULTS

The completed final achieves all of my goals for this project. Each microinteraction plays into the next, creating a sense of motion throughout the whole product. I'm excited to implement this into my portfolio and I think it will improve both the navigation of my site and its sense of style.
